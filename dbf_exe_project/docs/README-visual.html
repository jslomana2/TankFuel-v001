PROCONSI – Tanques · README visual

Vista general de la interfaz y pasos básicos para ejecutar el proyecto.

  1. Instala dependencias: pip install -r requirements.txt
  2. Ejecuta: python app.py y abre http://127.0.0.1:5000.
  3. Para EXE, empaqueta con PyInstaller y deja los DBF junto al EXE.
