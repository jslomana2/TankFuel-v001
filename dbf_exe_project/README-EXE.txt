USO RÁPIDO (EXE)

1) Ejecuta el workflow "Build Windows EXE" en GitHub.
2) Descarga el artefacto PROCONSI-Tanques.exe.
3) Crea una carpeta (p.ej. C:\Proconsi\Tanques\) y copia allí:
   - PROCONSI-Tanques.exe
   - FFALMA.DBF, FFARTI.DBF, FFTANQ.DBF, FFCALA.DBF
4) Doble clic al EXE y abre http://127.0.0.1:5000
   (Opcional) PROCONSI-Tanques.exe "C:\Ruta\A\DBF"
